Configuração de Dados
Escolha como o painel lê seus dados no Supabase.
Se você não tem tabela “principal”, o mais comum é:
  • 1 tabela única contendo tudo, e uma coluna dizendo se é Principal/Obras/Notes.
  • Ou 3 tabelas com nomes diferentes (ex.: dados_principal, tbl_obras...).
Modo A: 1 tabela única
Você escolhe a tabela e a coluna que define a “aba” (Principal/Obras/Notes).
Modo B: 3 tabelas
Você define a tabela de cada aba.
O que o autodetect faz?
Ele tenta consultar algumas tabelas “prováveis” (por amostra) e, se achar uma que responda, sugere ela. Isso não lista tabelas (Supabase não permite listar pelo anon key facilmente), mas já resolve 90% dos casos.
